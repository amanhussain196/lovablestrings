# String Art Generator

Turn your photos into amazing string art!

## How to use
1. Open `index.html` in any modern web browser (Edge, Chrome, Firefox).
2. Upload an image.
3. Pan and zoom to select the circular area you want to convert.
4. Adjust settings (Number of nails, lines, density).
5. Click **Generate** and watch the magic happen!

## Features
- **Privacy First**: All processing happens in your browser. No images are uploaded to any server.
- **Customizable**: Control the number of nails and strings.
- **High Resolution**: Downloads high-quality output.

## Troubleshooting
If you encounter issues, ensure you are using a modern browser. No internet connection is required after loading.
